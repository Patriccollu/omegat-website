Zusätzliche Ressourcen für OmegaT

Zusätzliche Ressourcen für OmegaT

OmegaT-Plug-ins

Tokenizer-Plug-in Ermöglicht die Erkennung von gebeugten Formen beziehungsweise Flexionen in den unscharfen Treffern und in Treffern im Glossar.

Dienstprogramme

"Aligner" Dienstprogramm zur Ausrichtung. Zur Verfügung gestellt von Dmitri Gabinski. Geschrieben in Python.
"Bligner" Dienstprogramm zur Ausrichtung. Zur Verfügung gestellt von Didier Briel. Geschrieben in Python/Perl.
TMXMerger Kommandozeilen-Skript in Java, um zwei oder mehrere TMX-Dateien zusammenzufügen. Zur Verfügung gestellt von Henry Pijffers.
TMXCleaner Kommandozeilen-Skript in Java, um "überflüssige" Übersetzungseinheiten aus einer TMX-Datei zu entfernen: löscht alle Übersetzungseinheiten ohne Zielsegment oder solche, bei denen das Zielsegment identisch mit dem Quellsegment ist. Zur Verfügung gestellt von Henry Pijffers.
Toxic (alle Versionen) Dienstprogramm, um Dateien des Trados-Tag-Editors (TTX-Dateien) in ein Zwischenformat umzuwandeln, welches das Übersetzen in OmegaT ermöglicht. Zur Verfügung gestellt von Marc Prior.
Toxic für Windows Windows-Version von Toxic.
Toxic für Linux Binäre Version von Toxic für Linux.
Toxic Skripts Skriptversionen von Toxic. Plattformübergreifend, erfordert Tcl/Tk.
Extspell Externe Rechtschreibkontrolle. Zur Verfügung gestellt von Marc Prior. Erfordert Tcl/Tk und Aspell (beide frei, alle Plattformen).
Stripper Skript zum extrahieren von Text für Übersetzungen von Reintext-Dateien (z.B. Computerquelltext) zum Überstezen in OmegaT. Zur Verfügung gestellt von Marc Prior, erfordert Tcl/Tk.
TKount Dienstprogramm, um Wörter zu zählen. Zur Verfügung gestellt von Marc Prior. Tcl/Tk, Abiword und pdftotext nötig. Wurde in Linux getestet.
Ondiloto ONline DIctionary LOokup TOol- Dienstprogramm zur Nutzung von Online-Wörterbüchern. Erfordert Linux, w3m und Tcl/Tk (beide werden normalerweise mit Linux bereitgestellt). Zur Verfügung gestellt von Marc Prior.
Wordfast TMX File Converter Konvertiert Translation-Memories, die von Wordfast im TMX-Format generiert wurden, zu einer Form, die von OmegaT gelesen werden kann. Dienstprogramm zur Ausrichtung. Zur Verfügung gestellt von Dmitri Gabinski.
stripIT macro Makro, mit welchem interne Tags aus TMX-Dateien entfernt werden können. Nur für veraltete OmegaT Dateien. Mittlerweile exportiert OmegaT TMX-Dateien sowohl mit, als auch ohne interne Tags. Zur Verfügung gestellt von Sonja Tomaskovic. 
OpenOffice.org Segmentation Macros Ermöglicht OmegaT, Sätze statt Paragraphen zu segmentieren. Nur als Referenz. Diese Funktion ist mittlerweile in OmegaT integriert. Zur Verfügung gestellt von Benjamin Siband.
ReadMe Beschreibung von Benjamin Sibands Segmentierungsmakros.
