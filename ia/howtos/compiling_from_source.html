Howto - Compiling OmegaT from the source code

Guida de OmegaT:
Compilar OmegaT a partir del codice primari

Iste guida explica como compilar OmegaT ex le codice origine.

OmegaT es supplite preste compilate pro un numero de platteformas, includente un version multe platteforma. Dunque le usatores non necessita de compilar OmegaT ipse. Compilar OmegaT es pertinente pro disveloppatores qui desidera facer modificationes lor proprie a su codice.

1. Discarga e installa le Java Development Kit (JDK) e Apache Ant

Varie programmas pote esser usate pro compilar de nove le codice primari post que tu ha modificate lo, ma inter le plus conveniente es Apache Ant. Il es alsi necesse le Java Development Kit (JDK).

dependente sur como tu installa le Java Development Kit, tu pote deber cambiar tu "JAVA_HOME" attributo proque illo functiona con Ant.

2. Compila le codice

Le nomines del plica indicate in basso es sol pro illustration; le numeros de version mutara.

Aperi un fenestra pro linea de commando (consola, fenestra DOS). Cambia plica al plica principal in le codice origine de OmegaT (le plica continente le file build.xml), e.g.

cd C:\My Documents\MyOmegat\OmegaT_2.2.3_Beta_Source

Ex intra iste plica, exeque Apache Ant per scriber le executabile ant. Il es possibile necessitar de scriber le percurso complete, e.g.:

C:\Program Files\apache-ant-1.7.0-bin\apache-ant-1.7.0\bin\ant

Si le codice se compila con bon successo, tu videra le message "BUILD SUCCESSFUL". Si non, essera monstrate messages de cernita que (on spera) providera alicun idea re ubi es le problema.

3. Lancea OmegaT

Si tu codice esseva compilate con bon successo, le OmegaT compilate es site in:

OmegaT_2.2.3_Beta_Source\dist

Ancora in tu fenestra de linea de commando, cambia iste plica, e.g. con:

cd C:\My Documents\MyOmegat\OmegaT_1.8.0_01_Beta_Source\dist

e lancea OmegaT con:

java -jar OmegaT.jar

4a. 1st exemplo de modification: cambiar le color del marcator del segmento

In le plica

src/org/omegat/gui/editor

aperi le file SegmentBuilder.java in un editor de texto.

Trova le lineas:

return Styles.createAttributeSet(fg, Styles.COLOR_GREEN, true, null);

e

b = Styles.COLOR_GREEN;

e reimplacia

COLOR_GREEN

con

COLOR_BLUE.

In le plica

src/org/omegat/util/gui

aperi le file Styles.java in un editor de texto.

Trova le linea

public static final Color COLOR_GREEN = new Color(192, 255, 192);

Reimplacia

COLOR_GREEN

con

COLOR_BLUE

e

192, 255, 192

con

225, 225, 255.

4b. 2nd exemplo de modification: augmentar le numero de concordantias partial displicate, ab cinque a dece

In le plica

/src/org/omegat/util

aperi le file OConsts.java in un editor de texto.

Trova le lineas:

/** Number of fuzzy matches to display */
public static final int MAX_NEAR_STRINGS = 5;

e reimplacia

= 5

con

= 10

Pro ambe le exemplos, compila e proba como describite supra.
