OmegaT

官方版本（由OmegaT企畫小組發行組建）

Sourceforge下載區 請在此下載所有官方版本
Windows版本（推薦） 1.7.3 Windows版本下載捷徑，內含Java JRE。若您使用Windows，這是目前運作最平順的推薦版本。

特殊版本（其他貢獻者推出的測試組建及一般組建）：

OmegaT 1.4.1 嘉泰羅尼語（西班牙）版 由Mikel L. Forcada提供
OmegaT 1.3.5 南非語版 由Samuel Murray提供南非語介面。文件仍為英文。
捷徑組建 特殊組建由Dmitri Gabinski提供，含額外的鍵盤捷徑。

第三方軟體（必備或強力建議）

Sun Java Windows, Linux, Solaris.
Macintosh OS X 僅含資訊。Mac OS X使用者應可發現系統已安裝Java Runtime environment。
OpenOffice.org StarOffice免費版本。使OmegaT能處理MS Word, Excel 及Powerpoint等格式檔案。
bitext2tmx 從以前翻譯過的原文與翻譯結果，產生翻譯記憶檔案的第三方比對工具
