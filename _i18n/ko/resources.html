OmegaT 관련 자료

문서

완전한 화면상의 사용자 설명서와 간단한 "빠른 시작" 설명서가 OmegaT와 함께 다양한 언어로 제공됩니다.

유틸리티

"Aligner" aligner utility 드미트리 가빈스키(Dmitri Gabinski) 제공, 파이썬으로 작성.
"Bligner" aligner utility 다이디얼 브리얼(Didier Briel) 제공. 파이썬과 펄로 작성
TMXMerger 두개 이상의 TMX(Translation Memory eXchange) 파일을 합치는 자바 명령줄(command-line) 스크립트. 헨리 피즈펄스(Henry Pijffers) 제공.
TMXCleaner TMX(Translation Memory eXchange)파일로 부터 "불필요한것들"을 제거하는 자바 명령줄(command-line) 스크립트. 번역해야 할 부분(target segment) 없이 모든 TU들을 지워줍니다. 또는 번역해야할 부분(target segment)이 번역된 부분(source segment)과 같은 곳에서 모든 TU들을 지워줍니다. TU가 뭔지 잘 모르겠습니다. 헨리 피즈펄스(Henry Pijffers) 제공.
Extspell Extspell 외부 문법 검사기. 마크 프라이어(Marc Prior) 제공. tcl/tk and Aspell 필요 (무료, 모든 플랫폼에서 사용 가능).
TKount TKount 문자데이터 갯수 셈하는 프로그램. 마크 프라이어(Marc Prior) 제공. tcl/tk, Abiword and pdftotext 필요. 리눅스에서만 실행해 보았음.
Wordfast TMX file converter Wordfast에서 만들어진 TMX 형식을 OmegaT에서 사용 가능하도록 변환 해주는 프로그램. 드미트리 가빈스키(Dmitri Gabinski) 제공.
stripIT macro TMX파일 내부 표식(tag)을 제거하는 매크로. 오직 OmegaT로부터 만들어진 TMX에만 사용 가능. OmegaT는 현재 TMX files을 이출(export) 할때 내부 표식(tag)의 유무를 선택 할 수 있습니다. 손자 토마느코빅(Sonja Tomaskovic) 제공.
OpenOffice.org segmentation macros OmegaT를 문장 단위가 아닌 문단 단위로 구분 할 수 있도록 해줍니다.참조만 하세요. 이 기능은 현재 OmegaT에서 제공 됩니다.벤자민 시밴드(Benjamin Siband) 제공.
ReadMe 벤자민 시밴드(Benjamin Siband)의 구분(segmentation) 매크로 설명
