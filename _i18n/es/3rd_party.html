Software de terceros

Software de terceros para utilizar con OmegaT

OpenOffice.org Versión libre de StarOffice. Permite a OmegaT el manejo de archivos en formatos MS Word, Excel y Powerpoint.
bitext2tmx Herramienta de alineación de terceros para producir archivos de memoria de traducción a partir de textos originales y sus traducciones.
Líneas de comando y procedimientos de Samuel Murray Descripciones de las líneas de comando / procedimientos de Samuel Murray y enlaces a los mismos. La mayoría de estas líneas de comando requieren MS Windows, aunque algunos procedimientos también prodrían funcionar en otros sistemas operativos.
Hay líneas de comando / procedimientos disponibles para hacer lo siguiente:
  • Saltar segmentos que contienen un cierto conjunto de caracteres
  • Extraer todo el texto fuente de los archivos del proyecto en un único archivo de texto largo
  • Crear un archivo TMX que contiene todas las porciones con origen y destino idéntico
  • Añadir términos al glosario durante la traducción con OmegaT
  • Añadir números a todos los segmentos para hacerlos todos "únicos"
  • Borrar todo el texto con excepción de las etiquetas, o todas las etiquetas exceptuando el texto
  • Comprobación de ortografía (varios procedimientos)
