Outros recursos OmegaT

Utilitários

Utilitário de alinhamento "Aligner" Contribuição de Dmitri Gabinski. Escrito em linguagem Python.
Utilitário de alinhamento "Bligner" Contribuição de Didier Briel. Escrito em Python/Perl.
TMXMerger Script Java para linha de comando que permite fusionar dois ou mais ficheiros TMX. Contribuição de Henry Pijffers.
TMXCleaner Script Java para linha de comando que remove unidades de tradução (TU) "supérfluas" de um ficheiro TMX: apaga todas as TU sem segmento-alvo ou aquelas em que o segmento-alvo é idêntico ao segmento-fonte. Contribuição de Henry Pijffers.
Extspell Corretor ortográfico externo Extspell. Contribuição de Marc Prior. Necessita de tcl/tk e Aspell (ambos Software Livre em todas as plataformas).
TKount Utilitário de contagem de texto TKount. Contribuição de Marc Prior. Necessita de tcl/tk, Abiword e pdftotext. Foi testado em Linux.
Conversor de ficheiro TMX do Wordfast Converte memórias de tradução em TMX (Wordfast) num formato aceite pelo OmegaT. Contribuição de Dmitri Gabinski.
stripIT macro Macro para remover etiquetas internas de ficheiros TMX. Apenas para ficheiros OmegaT "legacy"; OmegaT exporta agora ficheiros TMX com ou sem etiquetas internas. Contribuição de Sonja Tomaskovic. 
Macros de segmentação para OpenOffice.org Permite segmentar o texto por frase em vez de ser por parágrafo. Apenas para referência; esta função é agora fornecida no OmegaT. Contribuição de Benjamin Siband.
Leia-me (ReadMe) Descritivo dos macros de segmentação feitos por Benjamin Siband.
