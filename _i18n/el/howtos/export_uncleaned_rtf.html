Οδηγός τού OmegaT:
Εξαγωγή αρχείου μορφής "Trados uncleaned RTF" από το OmegaT

Αυτός ο οδηγός περιγράφει την διαδικασία δημιουργίας "ακαθάριστου RTF" από ένα έργο τού OmegaT. Αυτή η μορφή αρχείου ίσως διευκολύνει τους χρήστες άλλων εργαλείων CAT, ιδίως του Trados και του Wordfast Classic.

Σημεία που χρειάζονται προσοχή:

Αυτός ο συνδυασμός κώδικα/διαδικασίας δεν έχει ακόμη δοκιμαστεί εκτενώς (αν και οι πρώτες αναφορές είναι ενθαρρυντικές).

Η δέσμη ενεργειών διατίθεται μόνο για την πλατφόρμα Windows.

Επιτρέπει στους μεταφραστές μόνο να δημιουργούν (δηλ. να παραδίδουν) ακαθάριστα αρχεία τού Trados. Δεν τους επιτρέπει να εκτελούν μετάφραση σε δίγλωσσα αρχεία τα οποία έχουν λάβει από πελάτες τους.

1. Εισάγετε τα σημάδια τεμαχισμού τού Trados

Για να δημιουργήσετε ένα δίγλωσσο αρχείο τού OpenOffice.org με σημάδια τεμαχισμού τού Trados, κλείστε το έργο τού OmegaT κι εκτελέστε την παρακάτω δέσμη ενεργειών (απαιτεί Windows) στο αρχείο project_save.tmx τού έργου:

http://www.leuce.com/tempfile/omtautoit/uncleanifyTMX.zip

2. Δημιουργείτε τα έγγραφα αφίξεως

Δημιουργήστε τα έγγραφα αφίξεως ως συνήθως και μετατρέψτε το αρχείο σε μορφή Microsoft Word.

3. Εφαρμόζετε στυλ στα σημάδια τεμαχισμού

Στο Microsoft Word, εκτελέσετε την παρακάτω μακροεντολή, για να εφαρμοστούν τα σχετικά στυλ στο/στα έγγραφα αφίξεως:

Private Sub AddTw4winStyles()
On Error Resume Next
Dim sMark, sInt, sExt As Style
Set sMark = ActiveDocument.Styles.Add(Name:="tw4winMark", _
    Type:=wdStyleTypeCharacter)
With sMark.Font
  .Name = "Courier"
  .Size = 11
  .Hidden = True
  .Subscript = True
  .ColorIndex = wdViolet
End With
Set sInt = ActiveDocument.Styles.Add(Name:="tw4winInternal", _
    Type:=wdStyleTypeCharacter)
sInt.LanguageID = wdNoProofing
With sInt.Font
  .Name = "Courier"
  .Size = 11
  .ColorIndex = wdRed
End With
Set sExt = ActiveDocument.Styles.Add(Name:="tw4winExternal", _
    Type:=wdStyleTypeCharacter)
sExt.LanguageID = wdNoProofing
With sExt.Font
  .Name = "Courier"
  .Size = 11
  .ColorIndex = wdGray50
End With
End Sub
