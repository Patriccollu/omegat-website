Sumber Daya Lain Untuk OmegaT

Plugin OmegaT

Penting: supaya menggunakan plugin ini, Anda mungkin perlu memasang OmegaT versi terbaru.
Tokenizers plug-in Mengenal bentuk-bentuk perubahan kata pada padanan fuzi dan hit glosarium. Kiat yang menjelaskan tokenizers plug-in juga tersedia.
Plugin Alat Bahasa Memeriksa kesalahan bahasa (gaya, gramar, dll.).
Plugin Skripting Memungkinkan skrip eksternal mengakses fungsionalitas OmegaT. (Harap tidak bingung dengan fitur "Ekspor Teks", yang juga memungkinkan penggunaan skrip bersama OmegaT, tetapi pada level yang lebih mendasar).
Plugin Filter Okapi Menyediakan filter untuk menangangi format berkas lainnya, meliputi TTX Trados.

Utilitas

Utilitas perata "Aligner" Ditulis dalam Pyton. Kontribusi dari Dmitri Gabinski.
Utilitas perata "Bligner" Ditulis dalam Pyton/Perl Kontribusi dari Didier Briel.
TMXMerger Skrip command-line Java untuk menggabungkan dua atau lebih berkas TMX. Kontribusi dari Henry Pijffers.
TMXCleaner Skrip command-line Java untuk menghapus unit terjemahan "yang tiada gunanya" dari berkas TMX: menghapus semua UT tanpa segmen target, atau segmen target sama dengan segmen sumber. Kontribusi dari Henry Pijffers.
Toxic Utilitas untuk mengubah berkas TagEditor Trados (TTX) menjadi bentuk intermediat, sehingga bisa diterjemahkan di OmegaT. Koleksi skrip ini mungkin bermanfaat bagi penulis skrip yang memahami tcl/tk. Pengguna lain disarankan untuk memakai Plugin Filter Okapi. Toxic merupakan kontribusi dari Marc Prior.
Toxic untuk Windows Toxic versi exe untuk Windows.
Toxic untuk Linux Toxic versi biner untuk Linux.
Skrip Toxic Berbgai versi skrip Toxic. independen-platform; memerlukan tcl/tk
Stripper Skrip untuk mengekstrak teks untuk diterjemahkan di OmegaT dari berkas teks-datar (msl. kode sumber komputer). Memerlukan tcl/tk. Kontribusi dari Marc Prior.
TKount TKount, utilitas penghitung teks. Memerlukan tcl/tk, Abiword dan pdftotext. Sudah diuji di Linux. Kontribusi dari Marc Prior.
Ondiloto Alat Pencari Kamus Online. Memerlukan Linux, w3m dan tcl/tk (keduanya umumnya sudah tersedia bersama Linux). Kontribusi dari Marc Prior.
Koverter berkas TMX Wordfast Mengubah memori terjemahan dalam format TMX yang dihasilkan oleh Wordfast menjadi bentuk yang bisa dibaca oleh OmegaT. Kontribusi dari Dmitri Gabinski.
