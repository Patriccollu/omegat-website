OmegaT HowTo:
Google Translate gebruiken in OmegaT

Naast de webpagina waarin gebruikers eenvoudigweg tekst kunnen plakken om te laten vertalen, biedt Google Translate ook een interface (API) die kan worden gebruikt door CAT-programma's om toegang te krijgen tot de service. Vanaf december 2011 wil Google dat gebruikers betalen voor het gebruik van deze service. Deze HOWTO beschrijft hoe deze functie kan worden geconfigureerd in OmegaT.

1. Als u er nog geen hebt, open dan een account op google.com.

2. Volg deze koppeling om bij Google aan te vragen de sleutel voor de Google Translate API v2. Zorg er voor dat u uw details van uw creditcard bij de hand heeft. Google zal uw sleutel voor de API genereren, een tekenreeks met ongeveer 39 tekens.

3. Wijs de sleutel als volgt toe aan de startprocedure voor OmegaT, afhankelijk van of u Windows, Mac gebruikt of OmegaT start vanaf de opdrachtregel of door middel van een script:

Windows-gebruikers:

3.1. Lokaliseer het configuratiebestand OmegaT.l4J.ini (de standaard locatie is C:\Program Files (x86)\OmegaT).

3.2. U heeft misschien rechten als Administrator nodig, of moet u het tijdelijk verplaatsen naar een andere locatie op uw harde schijf om dit bestand te kunnen bewerken.

3.3. Open OmegaT.l4J.ini in een platte-tekstbewerker

3.4. Voeg de volgende regel toe aan de tekst in het bestand:

-Dgoogle.api.key=XXXXXXXX

(waarbij "XXXXXXXX" de sleutel voor de API is die Google heeft verschaft).

3.5. Sla OmegaT.l4J.ini op (en, indine nodig, verplaats het terug naar de map van OmegaT).

Mac-gebruikers:

3.6 Klik met rechts op OmegaT.app en selecteer "Pakketinhoud weergeven".

3.7 Klik met rechts op het bestand Info.plist en selecteer de bewerker van uw voorkeur om het bestand te openen.

3.8. Zoek de regels:

<key>apple.awt.graphics.UseQuartz</key>

<string>true</string>

en voeg direct daarna de regels toe:

<key>google.api.key</key>

<string>XXXXXXXX</string>

waarbij "XXXXXXXX" uw sleutel voor Google Translate API is.

3.9 Sla Info.plist op.

Linux-gebruikers:

3.10 Zorg er voor dat het Kaptain-startbestand is ge√Ønstalleerd (bekijk indien nodig de Linux Howto), en voer de Google Translate API-sleutel in het relevante veld in op de tab Connectivity.

Gebruikers die OmegaT starten vanaf de opdrachtregel of door een script:

3.11 Voeg het volgende argument toe aan de startopdracht:

-Dgoogle.api.key=XXXXXXXX

waarbij "XXXXXXXX" uw sleutel voor Google Translate API is.

De basis startopdracht voor de toegevoegde Google Translate API zal daarom zijn:

java -jar -Dgoogle.api.key=XXXXXXXX OmegaT.jar

4. Configuratie in OmegaT

4.1. OmegaT starten

4.2. Selecteer Opties > Machinevertaling

4.3. Vink Google Translate v2 aan (laat de andere opties niet aangevinkt)

5. Opmerking:

5.1 Google Machinevertaling is een betaalservice (US$20.00 per miljoen tekens). U kunt het dagelijks gebruik controleren op uw account van Google. In het bovenste menu:

More > Even More > Innovation/Code > APIs Console > Service/Translate API
