O proxecto OmegaT e vostede

OmegaT non é só un produto, é un apaixonante proxecto do cal pode ser parte.

OmegaT é software libre. Iso significa que os usuarios como vostede son libres de descargalo e usalo con moi poucas restricións. Tamén é libre de copialo e pasarllo a outra xente.

OmegaT é software de código aberto. Iso significa que pode modificar OmegaT para adaptalo ás súas necesidades se así o desexa. Para os defensores do software libre, esta liberdade é aínda máis importante que o feito de non ter que pagar nada.

O proxecto de OmegaT beneficiaríase de un maior apoio por parte de usuarios de todo o mundo. Inda que moitos usuarios son conscientes disto, a reacción habitual é \xABNon sei programar!\xBB Programar é sen lugar a dúbidas a tarefa máis importante, e sen a programación non hai programa. Pero programar non é o único traballo que debe realizarse. A continuación ten unha lista de áreas na que serían benvidos máis voluntarios!

Creación de documentación

A documentación é moi importante. Moitas das persoas que están lendo isto serán tradutores que terán lido moitos documentos de usuario. Pensou algunha vez en crear documentos de usuario? Cambiará a súa percepción!

Localización

Grazas ao traballo de tradutores no pasado, a interface de usuario de OmegaT e o seu manual foron traducidos a varios idiomas. Iso significa que só faltan uns cantos miles! Localizar OmegaT a outros idiomas é unha tarefa continua, xa que novas características e funcións son engadidas a OmegaT constantemente. Lea a Guía de localización de OmegaT para máis información sobre localizar OmegaT.

Facer probas

Unha das cousas máis útiles que poden facer os usuarios de OmegaT é... usar OmegaT e informarnos cando atopan cousas que non funcionan, ou diferencias na documentación.

Programar (Java)

OmegaT está escrito en Java. Se é un programador con coñecementos de Java, gustaríanos contactar con vostede. Se non...

Programar (sen Java)

... non significa que toda a programación teña que ser realizada en Java. Se pode programar noutro idioma, pode escribir código para apoiar o programa. Este código complementario xa se escribiu en Star Basic e Tcl/Tk, por exemplo. Podo escoitar como di que non pode programar en absoluto. Considerou aprender? Se xa está familiarizado cos ordenadores e escolle unha linguaxe de procesamento de scripts directa como Tcl/Tk, podería aprender por si mesmo nunhas poucas semanas o suficiente para escribir scripts simples pero útiles. O desenvolvemento profesional continuo é a actual expresión de moda!

Publicidade
