OmegaT:n lataaminen

Ohjeet lataamiseen

OmegaT:n voi ladata useana erilaisena versiona. Useimmat käyttäjät saavat sopivan version vain seuraamalla alla olevaa linkkiä ja napsauttamalla painiketta, jossa lukee "Download Stable version...". Kannattaa kuitenkin ensin lukea alla olevat ohjeet.

Alusta (käyttöjärjestelmä)

OmegaT:stä on saatavana versiot Linuxille, Mac OS X:lle ja eri Windows-versioille. Lataussivu tunnistaa käyttöjärjestelmän itse, joten jos OmegaT aiotaan asentaa koneelle, jolla se ladataan, versiosta ei tarvitse huolehtia. Jos kuitenkin OmegaT aiotaan asentaa eri tietokoneelle, valitaan sen käyttöjärjestelmän linkki.

JRE

Toimiakseen OmegaT tarvitsee tietokoneelle asennetun Java Runtime Environment (JRE) -ympäristön. Ladattaviin Windows ja Linux -paketteihin sisältyy oletuksena JRE, joka asentuu järjestelmään automaattisesti OmegaT:tä varten.

Jos sopiva JRE-versio on varmasti jo asennettu tietokoneeseen, voit ladata OmegaT-version ilman JRE:tä. Tällöin ohjelma latautuu nopeammin ja vie vähemmän tilaa kovalevyllä. Valitse linkki, jossa lukee "Java not included".

Vaikka JRE olisikin jo asennettuna tietokoneeseen, OmegaT:n JRE asentuu erilleen siitä eikä häiritse sitä. On siis aina turvallinen vaihtoehto asentaa JRE:n sisältävä OmegaT-versio. Ellet ole varma, onko sinulla sopiva JRE-versio, valitse JRE:n sisältävä versio (eli oletusversio).

Mac OS X -käyttäjät: JRE on oletuksena asennettua Mac OS X:ään, joten JRE:n sisältävää Mac OS X -versiota ei ole.

Linux-käyttäjät: Linuxin useimpiin jakeluversioihin sisältyy Java Runtime Environment, mutta monet näistä ovat avoimen lähdekoodin toteutuksia, jotka eivät toimi oikein OmegaT:n kanssa. Suosittelemme vahvasti käyttämään Sunin JRE-versiota, joka sisältyy oletuspakettiin.

Latauslinkki

Luettuasi tekstin yllä  napsauta tätä niin pääset lataussivulle ja voit ladata OmegaT:n vakioversion..

Muut versiot

Ladattava vakioversio (oletusversio) on viimeisin vakaa versio. Muitakin versioita on saatavana, muun muassa:

Lataussivustolta on saatavana myös kehitys- eli betaversio. Se sisältää uusia ominaisuuksia, joita nykyisessä vakaassa versiossa ei ole. Se on todennäköisesti kohtuullisen vakaa ja toimiva, mutta siinä saattaa olla joitakin virheitä. Erityisesti uusia ominaisuuksia koskeva dokumentaatio ei ole niin kattava kuin vakaiden versioiden dokumentaatio. Versio voi sopia käyttäjille, joilla on jo kokemusta OmegaT:stä. Beta-version saadaksesi napsauta linkkiä yllä, sitten linkkiä, jossa lukee "Browse all packages".

Edistyneempi kehitysversio eli alfa-versio on myös saatavana. Se ei todennäköisesti ole lainkaan niin vakaa kuin beta-versio. Sitä suositellaan ainoastaan asialle omistautuneille kehittäjille. Sen saa käyttöön aliversiokomennolla:

svn co https://omegat.svn.sourceforge.net/svnroot/omegat/trunk omegat

Myös OmegaT:n lähdekoodi on saatavana. Seuraa jälleen linkkiä "Browse all packages".

Dmitri Gabinskin tekemä erityiskokoonpano, jossa on ylimääräisiä näppäinoikopolkuja, on saatavana täällä. (Kokoonpano saattaa perustua vanhempaan OmegaT-kokoonpanoon.)
