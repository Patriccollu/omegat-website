Návod OmegaT:
Nastavení překladače MS Translator na Macu


- Přihlašte se do obchodu Microsoft Azure Marketplace:
http://datamarket.azure.com/

*Pokud ještě nemáte účet pro Azure Marketplace bude nejprve nutno si jej založit.

- Klikněte na volbu Můj účet v horní části stránky.
- Ve spodní části stránky uvidíte záznamy a hodnoty pro:

Hlavní klíč účtu / Primary Account Key
Identifikaci uživatele / Customer ID

Hlavní klíč účtu = microsoft.api.client_secret (je potřebný k nastavení přístupu k překladači Microsoft Translator)
Identifikace uživatele = microsoft.api.client_id je potřebný k nastavení přístupu k překladači Microsoft Translator)
(
- Přejděte k adresáři aplikace (prostě tam, kde máte nainstalováno OmegaT na svém Macu), pravý klik/cmd+klik/dotek dvou prstů na spouštěcí soubor OmegaT a vyberte „Zobrazit obsah balíčku“ / „View package contents“.
- Klikněte na adresář s obsahem a zálohujte/zkopírujte/zduplikujte nalezený soubor info.plist.
- Otevřete soubor info.plist ve svém textovém editoru a najděte část s vlastnostmi („Properties section“).
- Sem přidáte záznamy se svým Hlavním klíčem účtu pro Azure Marketplace a Identifikaci uživatele (zapamatujte si, že microsoft.api.client_id = vaše identifikace uživatele pro Azure Marketplace a microsoft.api.client_secret = váš hlavní klíč k účtu na Azure Marketplace):

 <key>Properties</key>
       <dict>
           <key>apple.laf.useScreenMenuBar</key>
           <string>true</string>
           <key>apple.awt.graphics.UseQuartz</key>
           <string>true</string>
          --> <key>microsoft.api.client_id</key>
          --> <string>A1234-5678-1234-5678-12345678901</string>
          --> <key>microsoft.api.client_secret</key>
          --> <string>AaBb1234CcDd5678aAbBcCdD1234eEfFgGhH1234</string>
       </dict>

***POVŠIMNĚTE SI: Šipky ve výše uvedeném příkladu slouží POUZE pro ilustraci. NEZADÁVEJTE je do reálných hodnot.

- Jakmile jste provedli výše zmíněné záznamy, zkontrolujte, že zadané údaje jsou správné a uložte a zavřete soubor info.plist.
- Spusťte OmegaT a povolte použití překladače MS Translator:
Možnosti > Strojový překlad ... a vyberte překladač, který chcete aktivovat.
- Začněte používat OmegaT.
