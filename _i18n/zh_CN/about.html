OmegaT项目

OmegaT项目由一个非正式的国际志愿者组织维护。其目的是开发，分发和支持使用OmegaT，一个开源翻译记忆软件应用程序。

该项目的主要成员是：

Keith Godfrey Original author of the OmegaT application
Didier Briel Project manager
Alex Buloichik Lead developer
Aaron Madlon-Kay Integration manager
Marc Prior Website manager
Vincent Bidaux Documentation manager
Kos Ivantsov Localization manager
