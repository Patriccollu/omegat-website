Додаткові ресурси для OmegaT

Плагін OmegaT

Плагін Tokenizer Уможливлює розпізнавання відмінків та флексій у неточних збігах та збігах у глосарії.

Утиліти

"Aligner" Службова програма для вирівнювання. Надана Дмитрієм Габінським. Написана на Python.
"Bligner" Службова програма для вирівнювання. Надана Дідієром Брилем. Написана на Python/Perl.
TMXMerger Програма інтрефейсу командного рядка на Java для об'єднання двох або більше файлів TMX. Надана Генрі Піджферсом.
TMXCleaner Програма інтерфейсу командного рядка на Java для видалення "зайвих" одиниць перекладу з файлу TMX: видаляє всі одиниці перекладу без цільового сегмента або ті, для яких цільовий сегмент ідентичний вихідному. Надана Генрі Піджферсом.
Toxic (усі версії) Утиліта для перетворення файлів редактора Trados-Tag (файли TTX) у проміжний формат, який дозволяє вам перекладати в OmegaT. Надана Марком Прайором.
Toxic для Windows Windows-версія Toxic.
Toxic для Linux Linux-версія Toxic.
Toxic Skripts Версії скриптів Toxic. Для всіх платформ, потрібні Tcl/Tk.
Extspell Зовнішній контроль орфографії. Надана Марком Прайором. Вимагає Tcl/Tk і Aspell (безкоштовні, для всіх платформ).
Stripper Скрипт для вилучення тексту для перекладу з файлів plain text (наприклад, з вихідного коду) для перекладу в OmegaT. Наданий Марком Прайором, вимагає Tcl/Tk.
TKount Службова програма підрахунку слів. Надана Марком Прайором. Необхідні Tcl/Tk, Abiword та pdftotext. Протестована в Linux.
Ondiloto Утиліта ONline DIctionary LOokup TOol для використання онлайн-словників. Вимагає Linux, w3m, і Tcl/Tk (обидві програми, як правило, вже є на Linux). Надана Марком Прайором.
Wordfast TMX File Converter Перетворює перекладацьку пам'ять, створену Wordfast у форматі TMX, у формат, який може прочитати OmegaT. Службова програма для вирівнювання. Надана Дмитрієм Габінським.
stripIT macro Макрос, який можна використовувати для видалення внутрішніх тегів з файлів TMX. Тільки для застарілих файлів OmegaT. На даний момент OmegaT експортує файли TMX як з внутрішніми тегами, так і без них. Надано Сонею Томашкович. 
OpenOffice.org Segmentation Macros Дозволяє OmegaT сегментувати речення замість абзаців. Тільки для довідки, оскільки ця функція вже інтегрована в OmegaT. Надано Бенджаміном Сібандом.
ReadMe Опис макросу Бенджаміна Сібанда.

Різноманітні ресурси

Навчальний курс Навчальний курс OmegaT. Містить навчальні матеріали для студентів, керівника курсів, а також підбірку демонстраційних файлів для перекладу.
Icons Альтернативні іконки для використання в OmegaT.
Демонстраційні тексти для перекладу Для тестування та демонстрації OmegaT.
