Über JRE

JRE steht für "Java Runtime Environment", also "Java-Laufzeitumgebung". JRE ist eine Software, die von SUN entwickelt wurde und die OmegaT braucht, um funktionieren zu können. 

Benutzer sind regelmäßig verunsichert, ob sie bereits eine passende JRE-Version auf ihrem Computer installiert haben und falls nicht, welche Version von JRE sie installieren sollen, wo sie diese finden und wie sie sie installieren können. Um dieses Problem zu umgehen, liefert das OmegaT-Projekt jetzt Versionen von OmegaT inklusive einer passenden Version von JRE. Das JRE, das im OmegaT-Paket mitgeliefert wird, wird lokal auf den Benutzercomputer installiert und kollidiert nicht mit anderen eventuell bereits installierten JREs oder Java-Installationen. Daher kann es sicher installiert werden.

Benutzer die sich sicher sind, dass sie bereits die passende JRE-Version installiert haben, können eine OmegaT-Version ohne JRE wählen. Der Vorteil hiervon ist ein deutlich kleineres Downloadpaket, das bei der Installation weniger Platz auf der Festplatte des Benutzers einnimmt.
