Guida di OmegaT:
Impostazione di MS Translator su Mac


- Accedere al Microsoft Azure Marketplace:
http://datamarket.azure.com/

*Se non si possiede già un conto Azure Marketplace sarà necessario prima registrarne uno.

- Fare clic sull'opzione "Il mio conto" nella parte superiore della pagina.
- Vicino al fondo si vedranno le voci e i valori per:

Chiave account primaria (Primary Account Key)
Identificativo cliente (Customer ID)

Primary Account Key = microsoft.api.client_secret (necessaria per configurare il proprio accesso a Microsoft Translator)
Customer ID = microsoft.api.client_id (necessario per configurare il proprio accesso a Microsoft Translator)

- Andare alla cartella delle applicazioni (o dove è posizionata l'installazione di OmegaT sul proprio Mac), fare clic col pulsante destro/cmd + clic/tap con due dita sull'eseguibile di OmegaT e selezionare "Mostra contenuto pacchetto".
- Fare clic sulla cartella Contenuto ed eseguire un backup/copia/duplicato del file info.plist.
- Aprire il file info.plist nell'editor di testo preferito e individuare la sezione Proprietà.
- Qui si dovranno aggiungere le voci dei valori di Azure Marketplace Primary Key Account e Customer ID (ricordare che microsoft.api.client_id = il proprio ID cliente di Azure Marketplace e microsoft.api.client_secret = il proprio Azure Marketplace Primary Key Account):

 <key>Properties</key>
       <dict>
           <key>apple.laf.useScreenMenuBar</key>
           <string>true</string>
           <key>apple.awt.graphics.UseQuartz</key>
           <string>true</string>
          --> <key>microsoft.api.client_id</key>
          --> <string>A1234-5678-1234-5678-12345678901</string>
          --> <key>microsoft.api.client_secret</key>
          --> <string>AaBb1234CcDd5678aAbBcCdD1234eEfFgGhH1234</string>
       </dict>

*** IMPORTANTE: le frecce sopra riportate sono SOLO a scopo illustrativo. Non includerle nelle voci reali.

- Una volta create le voci di cui sopra, verificare che i valori siano corretti, poi salvare e chiudere il file info.plist.
- Avviare OmegaT e attivare l'opzione MS Translator:
Opzioni > Traduzione automatica... e selezionare le opzioni da attivare.
- Iniziare a usare OmegaT.
