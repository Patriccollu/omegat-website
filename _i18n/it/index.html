Introduzione a OmegaT

OmegaT è un'applicazione di memoria di traduzione gratuita scritta in Java. È uno strumento progettato per traduttori professionisti. Essa non traduce per te! (Il software che fa questo è chiamato "traduttore automatico", e si dovrà cercare altrove). OmegaT ha le seguenti caratteristiche:

  • Concordanze parziali (fuzzy)
  • Propagazione delle concordanze
  • Elaborazione simultanea di progetti con più file
  • Uso simultaneo di più memorie di traduzione
  • Glossari utente con riconoscimento delle forme flesse
  • I formati di file dei documenti includono:
    Microsoft Word, Excel, Powerpoint (con estensione docx, xlsx, pptx)
    XHTML e HTML
    Open Document Format (LibreOffice, OpenOffice.org)
    MediaWiki (Wikipedia)
    Testo semplice
    ... e circa trenta altri formati di file
  • Supporto Unicode (UTF-8): può essere utilizzato con alfabeti non latini
  • Supporto per le lingue da destra a sinistra
  • Correttore ortografico integrato
  • Compatibile con altre applicazioni per memorie di traduzione (TMX, TTX, XML, XLIFF, SDLXLIFF)
  • Interfaccia di Google Translate
