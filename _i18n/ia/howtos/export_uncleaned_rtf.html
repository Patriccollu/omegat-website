Guida de OmegaT:
Exportar ex OmegaT al formato "RTF non mundate Trados"

Iste Guida explica como producer un file "RTF non mundate" de ex projecto OmegaT. Iste formato pote esser un practic formato de intercambio pro usatores de altere applicationes CAT, in particular Trados e Wordfast Classic.

Punctos a notar:

Iste combination codice/procedura non esseva ancora probate a fundo (ben que le reportos initial es promittente).

Le script es utilisabile sol pro le platteforma Windows.

Illo habilita solmente le traductores a generar (id es committer) files non mundate Trados. Illo non habilita illes a "traducer" le files bilingue supplite per lor clientes.

1. Insere le delimiters de segmento de Trados

Pro producer a file bilingue OpenOffice.Org continente le delimiters de segmento Trados, claude le projecto OmegaT, pois exeque le script AutoIt sequente (necessita Windows) sur le the file project_save.tmx del projecto:

http://www.leuce.com/tempfile/omtautoit/uncleanifyTMX.zip

2. Crea documentos final

Crea le documentos final como de costume e converte le file al formato Microsoft Word.

3. Applicar le stilos pro delimitar

In Microsoft Word, exeque le macro sequente pro applicar le stilos apposite al documento(s) final:

Private Sub AddTw4winStyles()
On Error Resume Next
Dim sMark, sInt, sExt As Style
Set sMark = ActiveDocument.Styles.Add(Name:="tw4winMark", _
    Type:=wdStyleTypeCharacter)
With sMark.Font
  .Name = "Courier"
  .Size = 11
  .Hidden = True
  .Subscript = True
  .ColorIndex = wdViolet
End With
Set sInt = ActiveDocument.Styles.Add(Name:="tw4winInternal", _
    Type:=wdStyleTypeCharacter)
sInt.LanguageID = wdNoProofing
With sInt.Font
  .Name = "Courier"
  .Size = 11
  .ColorIndex = wdRed
End With
Set sExt = ActiveDocument.Styles.Add(Name:="tw4winExternal", _
    Type:=wdStyleTypeCharacter)
sExt.LanguageID = wdNoProofing
With sExt.Font
  .Name = "Courier"
  .Size = 11
  .ColorIndex = wdGray50
End With
End Sub
