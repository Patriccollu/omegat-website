Guida de OmegaT:
Usar Google Translate in OmegaT

Ultra al pagina TTT ubi le usatores pote simplemente collar le texto pro le traduction, Google Translate provide anque un interfacie (API) que pote esser usate per le applicationes CAT, pro acceder le servicio. A mesura que de decembre 2011, Google ha requirite le pagamento ab usatores pro uso de iste servicio. Iste Guida describe como configurar le function in OmegaT.

1. Si tu jam non ha uno, aperi un conto con google.com.

2. Seque iste ligamine pro referer te a Google pro le clave API v2 de Google Translate. Face te secur que tu ha commode le detalios de tu carta de credito. Google generara tu clave API, un linea con approximativemente 39 characteres.

3. Assigna le clave al procedura de lanceamento de OmegaT como seque, dependente sur si tu es usante Windows, Mac, tu lancea OmegaT ab le linea de commando o per medio de un script:

Usatores de Windows:

3.1. Locar le file de configuration OmegaT.l4J.ini (le ubication de arrangiamento initial es C:\Program Files (x86)\OmegaT).

3.2. Pro rediger iste file, tu pote haber necessitate del modo Administrator, o de mover lo temporaneemente a un differente ubication sur tu disco dur.

3.3. Aperi OmegaT.l4J.ini in un editor de texto simple

3.4. Adde le linea sequente al texto in le file:

-Dgoogle.api.key=XXXXXXXX

(ubi "XXXXXXXX" es le clave API fornite per Google).

3.5. Salva OmegaT.l4J.ini (e si necesse, move lo al reverso al plica OmegaT).

Usatores de Mac:

3.6 Clicca con le dextero sur OmegaT.app e selige "Show Package Contents".

3.7 Clicca con le dextero sur le file Info.plist e selige tu editor ideal pro aperir le file.

3.8. Trova le lineas:

<key>apple.awt.graphics.UseQuartz</key>

<string>true</string>

e immediatemente post los, adde le lineas:

<key>google.api.key</key>

<string>XXXXXXXX</string>

where "XXXXXXXX" is your Google Translate API key.

3.9 Salva Info.plist.

Usatores de Linux:

3.10 Face te secur que tu ha le lanceator Kaptain installate (refere te si necessari al Guida Linux), deinde scribe le clave de Google Translate API in le pertinente campo sur le etiquetta Connexivitate?????.

Usatores que lancea OmegaT per le linea de commando o per un script:

3.11 Adde le argumento sequente al commando de lanceamento:

-Dgoogle.api.key=XXXXXXXX

where "XXXXXXXX" is your Google Translate API key.

Le commando basic de lanceamento con le Google Translate API in consequentia essera:

java -jar -Dgoogle.api.key=XXXXXXXX OmegaT.jar

4. Configuration in OmegaT

4.1. Lancea OmegaT

4.2. Selige Optiones > Traduction Automatic

4.3. Marca Google Translate v2 (lassante non marcate le altere optiones)

5. Nota:

5.1 Le traductor automatic Google Machine Translation es un servicio pagate (US$20.00 per million de characteres). Tu pote controlar le uso quotidian ab tu conto Google. In le menu superior:

More > Even More > Innovation/Code > APIs Console > Service/Translate API
