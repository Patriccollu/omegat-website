Вспомогательные программы

«Выравниватель» параллельного текста «Aligner» Автор: Дмитрий Габински (Dmitri Gabinski). Язык программирования: Python.
«Выравниватель» параллельного текста «Bligner» Автор: Дидье Брейль (Didier Briel). Язык программирования: Python/Perl.
TMXMerger Java-сценарий для командной строки, используемый для объединения нескольких TMX-файлов. Автор: Генри Пиджферс (Henry Pijffers).
TMXCleaner Java-сценарий для командной строки, используемый для удаления из TMX-файла«лишних» единиц перевода, т.е. тех, у которых нет целевого сегмента или в которых целевой сегмент совпадает с исходным. Автор: Генри Пиджферс (Henry Pijffers).
Extspell Внешний модуль проверки правописания. Автор: Марк Прайор (Marc Prior). Требуются программные пакеты tcl/tk и Aspell (оба свободные и доступны для всех платформ).
Stripper Скрипт для извлечения текста из простых текстовых файлов (например, из исходного кода программ) для перевода в OmegaT. Автор: Марк Прайор (Marc Prior); требует tcl/tk.
TKount Программа подсчёта объёма текста. Автор: Марк Прайор (Marc Prior). Требуются программные пакеты tcl/tk, Abiword и pdftotext. Тестировался в ОС Linux.
Ondiloto Программа поиска слов в онлайн-словаре. Требуются ОС Linux и программные пакеты w3m и tcl/tk (оба поставляются вместе с Linux). Автор: Марк Прайор (Marc Prior).
Конвертер TMX-файлов Wordfast Преобразовывает созданную в программе Wordfast память переводов в формате TMX в формат, поддерживаемый системой OmegaT. Автор: Дмитрий Габински (Dmitri Gabinski).
Макрос stripIT Макрос для удаления внутренних тегов в TMX-файлах. Подходит только для старых файлов OmegaT. Сейчас OmegaT экспортирует TMX-файлы с и без внутренних тегов. Автор: Соня Томашкович (Sonja Tomaskovic).  
Макрос сегментации OpenOffice.org Позволяет OmegaT сегментировать текст по предложениям, а не по абзацам. Только для справки. В OmegaT эта функция недоступна. Автор: Бенджамин Сибанд (Benjamin Siband).
ReadMe Описание макроса сегментации Бенджамина Сибанда.

Значки Набор альтернативных значков для OmegaT.
Образцы текста Для тестирования и демонстрации возможностей OmegaT.
