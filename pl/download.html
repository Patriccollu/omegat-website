Pobierz OmegaT

Pobierz OmegaT

Instrukcje dotyczące pobierania

Narzędzie OmegaT jest dostępne w kilku wersjach. Poniżej znajdują się bezpośrednie odnośniki, dzięki którym możesz pobrać odpowiednią wersję.

Dostępne są wersje na systemy Windows, Mac OS X i Linux. Zakładamy, że wiesz, jaki masz system operacyjny na swoim komputerze.

Użytkownicy systemów Windows i Linux mogą ponadto pobrać wersję ze środowiskiem JRE lub bez. Jeśli nie wiesz czy na Twoim komputerze jest zainstalowane odpowiednie środowisko JRE, pobierz wersję z tym środowiskiem. Nawet jeśli odpowiednie środowisko JRE jest już zainstalowane na Twoim komputerze, możesz spokojnie zainstalować wersję narzędzia OmegaT z tym środowiskiem. Aby dowiedzieć się więcej o środowisku JRE, kliknij tutaj.

Po pobraniu narzędzia OmegaT rozpakuj zawartość archiwum ZIP do odpowiedniego katalogu. Następnie przeczytaj instrukcje zawarte w pliku readme.txt (lub jego tłumaczeniu, np. readme_fr.txt dla języka francuskiego) i postępuj zgodnie ze wskazówkami.

Odnośniki

Dla użytkowników systemu Windows:

OmegaT dla systemu Windows - ze środowiskiem JRE Jeśli używasz systemu Windows, w którym nie jest zainstalowane odpowiednie środowisko JRE (lub nie wiesz czy jest ono zainstalowane), kliknij odnośnik po lewej stronie, aby pobrać narzędzie OmegaT ze środowiskiem JRE. Jest to wersja "z pełnym wyposażeniem".
OmegaT dla systemu Windows - bez środowiska JRE Jeśli używasz systemu Windows, w którym jest zainstalowane odpowiednie środowisko JRE, kliknij odnośnik po lewej stronie, aby pobrać narzędzie OmegaT.

Dla użytkowników systemu Mac:

OmegaT dla systemu Mac OmegaT dla systemu Mac OSX.

Dla użytkowników systemu Linux:

OmegaT dla systemu Linux - ze środowiskiem JRE Jeśli używasz systemu Linux, w którym nie jest zainstalowane odpowiednie środowisko JRE (lub nie wiesz czy jest ono zainstalowane), kliknij odnośnik po lewej stronie, aby pobrać narzędzie OmegaT ze środowiskiem JRE. Jest to wersja "z pełnym wyposażeniem".
OmegaT dla systemu Linux - bez środowiska JRE Jeśli używasz systemu Linux, w którym jest zainstalowane odpowiednie środowisko JRE, kliknij odnośnik po lewej stronie, aby pobrać narzędzie OmegaT.
