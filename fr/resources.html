Modules d'extensions pour OmegaT

Important : l'utilisation de ces modules peut exiger l'installation d'une version d'OmegaT plus récente.
Tokenizer Lemmatiseur d'OmegaT pour l'identification des flexions dans les glossaires et correspondances approximatives. Un guide pratique à propos du module Tokenizer est également disponible.
LanguageTool Vérificateur linguistique (erreurs de style, grammaire, etc.)
Scripting Module d'accès aux fonctionnalités d'OmegaT via des scripts externes (plus évolué que la fonction d'exportation de texte permettant également l'utilisation de scripts avec OmegaT)
Filtres Okapi Extension de gestion de formats de fichier supplémentaires, notamment Trados TTX.

Utilitaires

Aligner Utilitaire d'alignement écrit en Python. Contribution de Dmitri Gabinski.
Bligner Utilitaire d'alignement écrit en Python/Perl. Contribution de Didier Briel.
TMXMerger Script de ligne de commande en Java pour fusionner les fichiers TMX (deux ou plus). Contribution de Henry Pijffers.
TMXCleaner Script de ligne de commande en Java permettant d'effacer les unités de traduction « superflues » dans un fichier TMX : il peut s'agir de toutes les unités de traduction sans segment cible ou dont le segment cible est identique au segment source. Contribution de Henry Pijffers.
Toxic Utilitaire destiné à la conversion des fichiers TTX (Trados TagEditor) dans un format intermédiaire permettant de les traduire avec OmegaT. Cet ensemble de scripts s'adresse aux programmeurs ayant des connaissances en tcl/tk. Les autres utilisateurs sont invités à utiliser l'extension de filtres Okapi. Toxic est une contribution de Marc Prior.
Toxic pour Windows Version de Toxic pouvant être exécutée avec Windows.
Toxic pour Linux Version binaire de Toxic pour Linux.
Scripts Toxic Versions script de Toxic. Multiplateforme – tcl/tk requis.
Stripper Script pour l'extraction de texte à partir de fichiers de texte brut (par ex. code source informatique) permettant la traduction avec OmegaT – tcl/tk requis. Contribution de Marc Prior.
TKount Utilitaire de comptage de texte – tcl/tk, Abiword et pdftotext requis. Testé sur Linux. Contribution de Marc Prior.
Ondiloto ONline DIctionary LOokup TOol, utilitaire de recherche dans les dictionnaires en ligne – Linux, w3m et tcl/tk requis (ces derniers sont généralement fournis avec Linux). Contribution de Marc Prior.
Priorview Utilitaire Tcl/tk fournissant un aperçu de la mise en page formatée des fichiers traduits. Contribution de Marc Prior.
Wordfast TMX file converter Outil de conversion des mémoires de traduction au format TMX générées par le logiciel Wordfast dans une forme pouvant être lue par OmegaT. Contribution de Dmitri Gabinski.
